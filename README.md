# 対話形式で画像から背景を除去する　GoBackgroundEraser

## 概要

GoBackgroundEraserから簡単な操作で背景を除去するアプリです。  
画像を読み込んだ後、範囲指定をして更新で初回の背景除去をした後、  
背景指定や前景指定をして更新をするサイクルを繰り返して、  
目的の背景を除去した画像を保存します。  
保存した画像はpng形式で、背景は透明となります。
